## Description

Hoshi is a powerful SSH and Mosh client for iOS, built for developers who need reliable terminal access from their phone or tablet.

Powered by the Ghostty terminal engine with Metal rendering, Hoshi delivers a fast, native terminal experience with features designed for real work:

- Multiple simultaneous SSH and Mosh connections
- Live session thumbnails — see what each session is doing at a glance
- 2-finger swipe to instantly swap between sessions
- Customizable keyboard toolbar with Esc, Ctrl, Opt, Tab, and arrow keys
- Pinch-to-zoom font sizing
- Mosh support for roaming between Wi-Fi and cellular without dropping sessions
- Auto-reconnect for interrupted SSH connections
- tmux session auto-attach
- Password and key-based authentication

Whether you're tailing logs on a production server, editing config files, or monitoring systems on the go, Hoshi gives you the terminal you need, wherever you are.
